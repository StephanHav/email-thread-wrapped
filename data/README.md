# Data Folder

Export your email thread from your preferred client as a .mbox file and drop it in this folder.

Thunderbird: In case you are using Thunderbird.

1. Create a new folder 'wrapped'
2. Copy the email thread into 'wrapped'
3. Copy your replies into 'wrapped' (they will combine into a single thread in the new folder)
4. Go to menu > add-ons and themes > install ImportExportTools NG
5. Press the 3 dots on the thread in 'wrapped' and 'Export messages as ..' new .mbox file
6. Drop .mbox file in this folder
